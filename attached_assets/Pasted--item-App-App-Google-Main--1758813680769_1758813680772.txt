📱 해외여행자 가격 비교 App – 기획서 (MVP 버전)
1. 앱 핵심 기능 (MVP)

실물 인식 (Google Lens API 활용)

사용자가 카메라로 상품을 찍으면 → 상품명/모델명 추출.

현지 통화 가격 인식 (OCR 활용 → 이미지 내 Price 텍스트 자동 추출).

환율 자동 반영

Open Exchange Rates API 또는 Google Finance API 활용.

예: 100 THB → 3,800 KRW 자동 계산.

출신국가 온라인 최저가와 비교

한국 사용자 기준: 쿠팡, 네이버쇼핑, 무신사 등 오픈 API/크롤링.

비교 결과 즉시 표시:

예: 태국 현지 = KRW 10,000, 한국 = KRW 20,000 → +10,000원 이득.

재미 요소 (UX 메시지)

현지 구매가 훨씬 유리 → "다 쓸어담어!!! 🛒🔥"

거의 차이 없음 → "굳이 이걸 여기서? 🤔"

한국이 더 쌈 → "여행에 충실해라... 🏖️"

저장 및 소비 기록 기능

사용자가 [샀다] 버튼만 눌러주면 → “오늘 총 절약 금액” 기록.

누적 그래프 제공 → "비행기값 벌어가기 챌린지 ✈️💸"

2. 확장 기능 (2차 버전 이후)

위치 기반 큐레이션: 같은 상품의 최저가 지도 표시 (ex. “치앙마이 시장보다 방콕 MBK가 더 쌈”).

상품 카테고리 추천: 현지에서 꼭 사야 하는 베스트 아이템 자동 큐레이션.

소셜 기능: 여행자들끼리 후기 공유 → “이거 여기서 득템했어요!”

3. 기술 스택 (Replit 개발 기준)

Frontend: React (PWA 기반 → 웹앱 & 앱스토어 동시에 대응).

Backend: Node.js + Express (가격 비교 API, 환율 API 연동).

Database: Firebase or Supabase (사용자 기록, 절약 금액 저장).

API 활용:

Google Lens API (비전 AI + OCR)

Open Exchange Rates (환율)

네이버 쇼핑/쿠팡 오픈 API (한국 가격 비교)

배포: Replit → Vercel/Netlify + PWA.

4. 초기 마케팅 전략

Target 사용자

한국 여행객 (태국·일본·베트남·대만 등 단기 여행 인기 국가).

특히 20~40대 여성/커플/가족 (쇼핑 중심 여행).

진입 전략

여행자 커뮤니티: 네이버 카페, 페이스북 여행 그룹, 인스타 릴스, 유튜브 쇼츠 활용.

인플루언서 체험단: 여행 유튜버/쇼핑 인스타그램 계정에게 무료 체험 제공.

공항 광고/리플렛: 인천공항·김해공항 출국장에 “현지 쇼핑 득템 앱” 홍보.

수익화

Affiliate (쿠팡 파트너스, 네이버 애드포스트).

Premium 구독: 가격 알림, 구매 이력 엑셀 다운로드, 가족 공유 기능.

제휴 여행사/면세점 광고.

5. 해외 확장 전략

국내 버전 → 인바운드 고객 대상

한국 inbound (일본·대만·홍콩 관광객): 한국에서 “롯데, 신라, 무신사 가격 비교” 서비스.

한국은 쇼핑 관광객이 많기 때문에 Inbound 버전은 성공 확률 높음.

차기 타겟 국가

태국: 한국인/중국인 관광객 대상 (쇼핑 비중 높음).

일본: 전자제품·패션 중심 (다국적 관광객).

대만/베트남: 한국인 여행객 급증 지역.

6. 추가 제안 (차별화 요소)

영수증 OCR 기능: 실제 결제 금액 스캔 시 자동 반영.

절약 순위 랭킹: “이번 달 여행 중 1등은 방콕에서 48만원 절약한 OOO님!”

현지 추천 메시지: “이 상품은 한국보다 30% 싸요. 현지 한정판 주의!”
